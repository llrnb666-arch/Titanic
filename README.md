# Titanic survival prediction

本项目不只是一个简单的kaggle泰坦尼克号预测入门

作为一名具备C语言与高等数学背景的开发者，我在这份notebook中，剥离了TFDF的高级API封装，用底层的理工科思维重新解构了机器学习的训练过程

## 核心拆解（详见代码批注）
1. **内存与数据结构映射：** 将 Pandas 的数据清洗操作映射为 C 语言的指针防越界与深拷贝保护；将 `remove` 特征选择映射为顺序表的内存移位。
2. **微积分与负梯度寻优：** 揭示 GBT（梯度提升树）并不是“投票”，而是一场基于 Loss 函数偏导数的“残差拟合接力赛”。
3. **空间解析几何的应用：** 解释 `SPARSE_OBLIQUE`（倾斜分割）如何突破传统轴对称二叉树的局限，在多维空间寻找最优分类超平面。
4. **算力对抗方差（Ensemble）：** 在模型集成阶段，通过循环动态改变 `random_seed` 注入伪随机扰动，用 100 个“和而不同”的专家森林投票，从统计学层面消除单一模型的过拟合偏见。

这份代码记录了我从“调包算法”向“底层工程架构”认知的跨越。
